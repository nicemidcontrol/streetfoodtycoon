# SFT — Master Plan & Milestones (v0.1)

## Milestone A — Pre-Production
**Deliverables**
- 1-page GDD (ไฟล์นี้)
- Art Bible (palette/scale/tile rules)
- รายการ “ห้ามทำตอนนี้” กันสโคปบวม

**DoD**
- ตอบได้ใน 10 วิว่าเกมเล่นยังไง และ MVP มีอะไรบ้าง

---

## Milestone B — Vertical Slice
**เป้าหมาย:** มีชิ้นเล็ก ๆ ที่ “เหมือนเกมจริง”
- แผงขาย 1 จุด + ลูกค้าเดิน + ขาย 1 เมนู
- เงินเพิ่ม + ปุ่มอัปเกรด 1 อย่าง (เช่น เตา)
- UI ขั้นพื้นฐาน (HUD เงิน + ปุ่มอัปเกรด)
- Save/Load แบบ Local

**DoD**
- เล่น 3 นาทีแล้วเข้าใจเอง

---

## Milestone C — MVP Alpha
- 3 เมนู + สูตรต้นทุน/กำไร
- พนักงาน (เพิ่มความเร็ว/ทำงานอัตโนมัติ)
- Progression: Unlock อย่างน้อย 1 อย่าง (เมนู/โซนย่อย)
- Offline earnings (เพดานเช่น 4 ชม.)
- Tutorial สั้น ๆ 3–5 ขั้น
- Balancing pass รอบแรก

**DoD**
- เล่นได้ 30–60 นาทีแบบไหล ๆ มีเป้าหมายต่อ

---

## Milestone D — MVP Beta (พร้อมให้เพื่อนเทสต์)
- Polish UI/UX + feedback/animation เล็ก ๆ
- Bug fix + performance pass
- Firebase Analytics events (ขั้นต่ำ)
- Remote Config (ปรับบาลานซ์โดยไม่ต้องอัปเดตแอป)
- Android build สำหรับทดสอบ 1–2 สัปดาห์

**DoD**
- ได้ feedback ชัดเจนว่า “ติดตรงไหน/งงอะไร/อยากเล่นต่อไหม”

---

## Milestone E — Soft Launch
- Daily reward / simple quest (เบา ๆ)
- Economy re-balance จากข้อมูลจริง
- Rewarded Ads only (ยังคงไม่กวน)
- Crash/ANR monitoring + hotfix plan

**DoD**
- Retention พอไหว + ไม่มีบั๊กพังเกม

---

## Milestone F — Global Launch + Content Pipeline
- เพิ่มสถานที่/เมนู/อีเวนต์ด้วยแพทเทิร์นเดิม
- event rotation / live ops เบื้องต้น
- cosmetic monetization (ตกแต่งร้าน/สกิน)

---

## Sprint Suggestion (สำหรับทำคนเดียว)
- Sprint 1: Pre-Production + project skeleton
- Sprint 2: Vertical Slice playable
- Sprint 3: MVP Alpha systems
- Sprint 4: Beta polish + Firebase + playtest build
- Sprint 5: Rebalance + soft launch readiness
